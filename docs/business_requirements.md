# Business Requirements

## Project Title

**Olist E-commerce Sales Analytics**

---

## Problem Statement

Olist is a Brazilian e-commerce marketplace that connects customers with multiple sellers. The business needs a data analytics solution to understand sales performance, customer behavior, delivery efficiency, payment trends, and seller performance.

---

## Project Objectives

- Design a relational database using PostgreSQL.
- Import and manage raw e-commerce data.
- Perform data quality validation.
- Analyze business performance using SQL.
- Build an interactive Tableau dashboard.
- Generate actionable business insights.

---

## Business Questions

- What is the total revenue generated?
- Which product categories generate the highest sales?
- Which sellers contribute the most revenue?
- Which states have the highest number of customers?
- How long does delivery take on average?
- Which payment methods are most frequently used?
- What is the customer review distribution?
- Which months have the highest sales?

---

## Expected Deliverables

- PostgreSQL Database
- Entity Relationship Diagram (ERD)
- SQL Analysis Scripts
- Tableau Dashboard
- Business Insights Report
- GitHub Documentation
