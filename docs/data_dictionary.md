# Data Dictionary

| Table | Description |
|--------|-------------|
| customers | Customer information |
| orders | Order details and delivery timeline |
| order_items | Products included in each order |
| products | Product information |
| sellers | Seller information |
| order_payments | Payment details |
| order_reviews | Customer reviews and ratings |
| geolocation | ZIP code geographical information |
| product_category_translation | Portuguese to English product category mapping |

---

## Key Fields

| Field | Description |
|--------|-------------|
| customer_id | Unique customer identifier |
| order_id | Unique order identifier |
| product_id | Unique product identifier |
| seller_id | Unique seller identifier |
| review_id | Unique review identifier |

---

## Data Source

Dataset: **Olist Brazilian E-commerce Public Dataset**

Source: Kaggle

Records: Approximately 100,000 orders across multiple related tables.
